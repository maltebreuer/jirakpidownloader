# jirakpidownloader

Extracts specific Scrum-oriented KPIs from a running JIRA instance.

KPIs are:

- velocity (planned vs. actual results of sprints)
- all issue data of a specific JIRA project
- some JIRA master data (issue types, status)

## Prerequisites

- JIRA Software (version 7+)

## Known issues

- custom fields are not aligned with every server version
